# api
install Nodejs<br>
install Loopback 4 CLI `npm install -g @loopback/cli`<br>
git clone <br>
npm install <br>
Create database [blog]<br>
update db details in src/datasources/mysql.datasource.config.json<br>
npm run build<br>
num run migrate<br>
npm start - to start the application<br>
